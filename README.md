# Bokado - Frontend

Це фронтенд частина комплексного курсового проєкту **Bokado**.

## 🔧 Використані технології

- **React** — для створення інтерфейсу користувача  
- **TypeScript** — для типізації та надійності коду  
- **Vite** — як сучасний збирач для швидкої розробки  
- **Redux Toolkit** — для управління глобальним станом  
- **Axios** — для взаємодії з бекендом через HTTP-запити

## 📁 Опис

Цей репозиторій відповідає за реалізацію клієнтської частини застосунку **Bokado**, що працює у зв'язці з бекенд-частиною, яка розміщена в окремому репозиторії.
